==============================================================================
【ソフト名】
	DribbbWall
【動作環境】
	.NET Framework 4.0以上
==============================================================================
【ソフト紹介】
Dribbble(http://dribbble.com/)でLikeした画像を取得して
デスクトップの壁紙に設定するアプリケーションです。

==============================================================================
【インストール】
・SetUp.msiをダブルクリックし、ウィザードの指示に従ってインストールを行なってください。
==============================================================================
【アンインストール】
・アプリケーションの追加と削除から[DribbbleWall]アンインストールを行なってください。
